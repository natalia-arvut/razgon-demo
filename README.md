# РАЗГОН — демо сайта цифрового агентства

Демо главной страницы корпоративного сайта-визитки веб-студии с лидогенерацией.
Подготовлено как пример под ТЗ заказчика (Kwork).

**Концепция:** светлый минимализм в духе cuberto.com + кинетические «вау»-эффекты
из стиля locomotive.ca (скрэмбл-типографика, видео-обложки кейсов), на чистом коде.

## Что реализовано
- Светлая тема, крупная типографика (Manrope), графитовые акценты — по ТЗ.
- Вау-элементы: кастомный курсор, магнитные кнопки, кинетическая (скрэмбл) типографика,
  scroll-анимации появления, плавный скролл, видео-обложки кейсов, переход-лоадер.
- Структура главной по ТЗ: hero (УТП) → 5 услуг → 5 кейсов → обо мне → отзывы →
  FAQ-аккордеон → форма заявки (Имя, Телефон, согласие 152-ФЗ) → футер.
- Адаптив, бережно к `prefers-reduced-motion`, контент доступен и без JS.

## Стек демо
HTML / CSS / JS + GSAP (анимация). Боевая версия — на Next.js + Headless CMS, хостинг в РФ.

## Заглушки
Тексты, кейсы, отзывы и фото основателя — структурные «рыбы»; заменяются на реальные
данные заказчика на этапе наполнения. Видео-обложки — свободная лицензия Pexels.

Логотип и палитра — `logo.html`.
